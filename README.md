### Авторизация
1. принимаем логи и пароль
2. сверяем их с бд
3. отправляем ответ на фронт(ok/no. 

### Генерация токена
1. принимаем открытый ключ(1. 
2. генерируем токен, с жизниным циклом(20-30 мин. 
3. сохраняем этот токен в бд(20-30 мин. 
4. шифруем токен принятым открытым ключем
5. генерируем новые открытый и закрытый ключ(2. , сохраняем их в бд также на (20-30 мин. 
6. отправляем на фронт шифрованый токен ключем(1.  + отправляем новый открытй ключ(2. 

### Запрос на перевод
1. принимаем зашифрованный токен ключем(2. (вытащить из json. 
2. дешифруем его
3. сверяем с токеном из бд
4. отправляем ответ на фронт(ok/no.
